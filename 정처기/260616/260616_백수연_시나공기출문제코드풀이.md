# 정처기 실기 공부

### 02 Java 문제 1 - 예제 1.java

```java
class ClassA {
	int a = 10;
	int funcAdd(int x, int y) {
		return x + y + a;
	}
}
public class Test {
	public static void main(String[] args) {
		int x = 3, y = 6, r;
		ClassA cal = new ClassA();
		r = cal.funcAdd(x, y);
		System.out.print(r);
	}
}
```

```c
// 예상 정답
19
```

<aside>

**내 풀이**

ClassA는 10의 값이 할당된 int형 멤버변수 a와 2개의 매개변수를 받아 멤버변수 a와의 총합을 출력하는 funcAdd 함수로 구성

ClassA cal = new ClassA(); 를 통해 ClassA형 변수 cal에 ClassA를 동적할당한다.
r = cal.funcAdd(x, y) = 3 + 6 + 10 = 19 이다.

</aside>

### 03 Java 문제2 - 예제 1.java

```java
class ClassA {
	ClassA() {
		System.out.print('A');
		this.prn();
	}
	void prn() {
		System.out.print('B');
	}
}
class ClassB extends ClassA {
	ClassB() {
		super();
		System.out.print('D');
	}
	void prn() {
		System.out.print('E');
	}
	void prn(int x) {
		System.out.print(x);
	}
}
public class Test {
	public static void main(String[] args) {
		int x = 7;
		ClassB cal = new ClassB();
		cal.prn(x);
	}
}
```

```c
// 진짜 정답
AED7
```

<aside>

**오답 풀이**

이 문제는 생성자 실행 중에도 오버라이딩된 메서드가 호출된다는 것을 보여주는 문제
즉 ClassB 객체를 만들면 자식 생성자인 ClassB가 실행되기 전에 부모 생성자인 ClassA 실행

class ClassB extends ClassA = ClassB는 ClassA를 상속받는다
ClassA = 부모 클래스, 상위 클래스, superclass
ClassB = 자식 클래스, 하위 클래스, subclass

클래스 명과 동일하게 정의된 ClassA와 ClassB는 각 클래스의 생성자, 할당될 때 자동 실행

super(): 부모 클래스의 생성자를 호출하는 문장

즉 최종 흐름은
ClassB cal = new ClassB()
→  ClassB 생성자 시작 → super() → ClassA 생성자 시작 → ‘A’ 출력 → this.prn(); → ClassB의 prn() → ‘E’ 출력 → ClassB의 super() 실행 끝 → ‘D’출력 → cal.prn(x); 실행 → 7 출력

</aside>

### 기출 따라잡기 - 문제 2.java

```java
public class Test {
	static int[] arr() {
		int a[] = new int[4];
		int b = a.length;
		for(int i = 0; i < b; i++)
			a[i] = i;
		return a;
	}
	public static void main(String[] args) {
		int a[] = arr();
		for(int i = 0; i < a.length; i++)
			System.out.print(a[i] + " ");
	}
}
```

```c
// 예상 정답
0 1 2 3 
```

<aside>

**내 풀이**

static int[] arr()는 {0, 1, 2, 3}이 할당된 배열을 반환하므로 반복문에서는 0 ~ 3까지 출력됨

</aside>

### 기출 따라잡기 - 문제 3.java

```java
class A {
	int a;
	public A(int a) { this.a = a; }
	void display() { System.out.println("a=" + a); }
}
class B extends A {
	public B(int a) {
		super(a);
		super.display();
	}
}
public class Test {
	public static void main(String[] args) {
		B obj = new B(10);
	}
}
```

```c
// 정답
a=10
```

<aside>

**오답 풀이**

B obj = new B(10); → super(10) → A(10) → this.a = 10 → A의 int a = 10 → super.display(); → void display() → System.out.println("a=" + a); → a=10 출력

B는 A를 상속 받았으므로 A의 안에 정의된 멤버 변수 a도 가지고 있음
즉, this가 할당될 때의 객체인 B를 의미하지만 상속으로 인해 변수 a에 할당도 되는 것

</aside>

### 기출 따라잡기 - 문제 4.java

```java
class Test {
	public static void main(String args[]) {
		cond obj = new cond(3);
		obj.a = 5;
		int b = obj.func( );
		System.out.print(obj.a + b);
	}
}

class cond {
	int a;
	public cond(int a) {
		this.a = a;
	}
	public int func( ) {
		int b = 1;
		for (int i = 1; i < a; i++)
			b += a * i;
		return a + b;
	}
}
```

```c
// 예상 정답
61
```

<aside>

**내 풀이**

new cond(3) → this.a = a → a = 3 → obj.a =5 → b = obj.func()
→ b = 1
→ b = b + a * 1 → b = 1 + 5 * 1 = 6
→ b = 6 + 5 * 2 = 16
→ b = 16 + 5 * 3 = 31
→ b = 31 + 5 * 4 = 51
→ a + b = 5 + 51 = 56
→ obj.a + b = 5 + 56 = 61

</aside>

### 기출 따라잡기 - 문제 5.java

```java
class A {
	int a;
	int b;
}

public class Test {
	static void func1(A m) {
		m.a *= 10;
	}
	static void func2(A m) {
		m.a += m.b;
	}
	public static void main(String args[]) {
		A m = new A();
		m.a = 100;
		func1(m);
		m.b = m.a;
		func2(m);
		System.out.printf("%d", m.a);
	}
}
```

```c
// 예상 정답
2000
```

<aside>

**내 풀이**

m.a = 100 → func1(m) → m.a *= 10 → m.a = 1000 → m.b = m.a → m.b = 1000 → func2(m) → m.a += m.b → m.a = 1000 + 1000 = 2000 → 2000 출력

</aside>

### 기출 따라잡기 - 문제 6.java

```java
class Parent {
	int x, y;
	Parent(int x, int y) {
		this.x = x;
		this.y = y;
	}
	int getX() {
		return x*y;
	}
}
class Child extends Parent {
	int x;
	Child(int x) {
		super(x+1, x);
		this.x = x;
	}
	int getX(int n) {
		return super.getX() + n;
	}
}
public class Main {
	public static void main(String[] args) {
		Parent parent = new Child(10);
		System.out.println(parent.getX());
	}
}
```

```c
// 예상 정답
110
```

<aside>

**내 풀이**

Parent parent = new Child(10);
→ super(x+1, x); → super(11, 10); → Parent(int x, int y) → Parent(int 11, int 10)
→ this.x = x; this.y = y; → x = 11, y = 10
→ this.x = x; → Child 지역변수 x = 10
→ System.out.println(parent.getX()); → parent.getX() → x*y = 11 * 10 = 110

</aside>
