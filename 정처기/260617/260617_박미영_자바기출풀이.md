### == 연산자
- 기본 데이터 타입(ex. int, float, char 등)의 값을 비교할 때 자주 사용됨.
- **두 값이 같은지 확인**
- 참조 데이터(ex. 객체)에 대해서 ==를 사용하면, 그것들이 **같은 메모리 주소를 가리키고 있는지** 확인
```
String text1 = new String("hello");
String text2 = new String("hello");
System.out.println(text1 == text2);

>>> false  # 왜냐하면 메모리 주소가 다르기 때문
```

### equals 메서드

- **객체의 내용이 같은지**를 확인하는 데 사용됨.
```
String text1 = new String("hello");
String text2 = new String("hello");
System.out.println(text1.equals(text2));

>> true  # 내용이 같기 때문
```

### [기출 예제] 23년 2회 실기

- **다음은 자바에 대한 문제이다. 알맞은 출력 값을 작성하시오.**

```python
public class Main{
		public static void main(String[] args) {
				String str1 = 'Programming';
				String str2 = 'Programming';
				String str3 = new String('Programming');
				
				println(str1==str2)
				println(str1==str3)
				println(str1.equals(str3))
				print(str2.equals(str3))
		}
}
```

- 정답
    - true
    - false
    - true
    - true
- 풀이
    - `String str1 = 'Programming';` 랑 `String str2 = 'Programming'` 는 하나의 객체가 생성되고 자바에서 같은 객체를 할당해줌.
    - `String str3 = new String('Programming');`
        - 명시적으로 `new`를 통해 객체를 생성하는 경우, 같은 값이더라도 **별도의 공간에 객체를 생성해 str3에 할당**함.

### [기출 예제] 22년 2회 실기

- **아래 자바 코드에서 출력되는 값을 작성하시오.**

```python
class Main {
		public static void main(String[] args) {
				Cond obj = new Cond(3);
				obj.a = 5;
				int b = obj.func();
				System.out.print(obj.a + b);
		}
}

class Cond {
		int a;
		public Cond(int a) {
				this.a = a;
		}
		
		public int func() {
				int b = 1;
				for (int i = 1; i < a; i++) {
						b += a * i;
				}
				return a + b;
		}
}
```

- 정답
    
    61
    
- 풀이
    
    ```python
    class Main {
    		public static void main(String[] args) {  # 1. 실행 시작
    				Cond obj = new Cond(3);  # 2. 객체 생성 => Cond 생성자 실행 => obj.a = 3
    				obj.a = 5;  # 5. obj.a 값 바꿈
    				int b = obj.func();  # 6. func() 호출, func() 안에서 a는 5 => b = 56
    				System.out.print(obj.a + b);
    		}
    }
    
    class Cond {
    		int a;
    		public Cond(int a) {  # 3. 실행 시작
    				this.a = a;  # 4. 객체의 a = 매개변수 a
    		}
    		
    		public int func() {
    				int b = 1;  # 7. 처음에 지역변수 b는 1
    				for (int i = 1; i < a; i++) {  # 8. a = 5니까 조건은 i < 5
    						b += a * i;  # b = 1 => 6 => 16 => 31 => 51
    				}
    				return a + b;  # 9. 5 + 51
    		}
    }
    ```
