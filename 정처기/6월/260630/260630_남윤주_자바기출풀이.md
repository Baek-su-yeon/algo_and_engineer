```java
class SuperObject {
    public void draw() {
        System.out.println("B");
        draw();
    }
    public void paint() {
        System.out.print('B');
        draw();
    }
    class SubObject extends SuperObject {
        public void paint() {
            super.paint();
            System.out.print('C');
            draw();
        }
        public void draw() {
            System.out.print('D');
        }
    }
    public class Test {
        public static void main(String[] args) {
            SuperObject a = new SubObject();
            a.paint();
            a.draw();
        }
    }
}
```
결과: BDCDD
SubObject 클래스의 생성자를 이용하여 SuperObject 클래스의 객체 변수 a를 생성.

a의 paint 메소드 호출. SubObject클래스의 paint() 메소드가 호출됨
