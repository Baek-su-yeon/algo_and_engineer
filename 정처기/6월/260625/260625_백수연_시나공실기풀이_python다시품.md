### 02 python 문제1 - 예제 1.py

```python
x, y = input('입력 :').split('-')
a = [ 'abc123', 'def456', 'ghi789' ]
a.append(x)
a.append(y)
a.remove('def456')
print(a[1][-3:], a[2][:-3], sep = ',')
for i in range(3, 6):
	print(i, end = ' ')
```

```python
# 출력
# 입력이 jkl123-mno456 이라고 가정
789,jkl
3 4 5
```

<aside>

x = jkl123, y = mno456

a = [ 'abc123', 'def456', 'ghi789', 'jkl123', 'mno456' ]
→ a = [ 'abc123', 'ghi789', 'jkl123', 'mno456' ]
→ a[1][-3:] = ghi789 → 789
→ a[2][:-3] = jkl123 → jkl (-3이라고해서 거꾸로 가는거 아님 주의, 걍 -3전에 끊으면 됨)
→ 789,jkl

for 문은 range(3, 6)이니까 3, 4, 5까지 출력

</aside>

### 03 python 문제2 - 예제1.py

```python
a = {'apple', 'lemon', 'banana'}
a.update( {'kiwi', 'banana'} )
a.remove('lemon')
a.add('apple')
for i in a:
	print("과일명 : %s" % i)
```

```python
# 출력
# set은 순서가 없는 자료형이라 출력 순서 보장 X
과일명 : apple
과일명 : banana
과일명 : kiwi
```

<aside>

add()는 원소 하나만 추가할 때 사용

update()는 여러 개의 원소를 각각 하나씩 꺼내서 추가할 때 사용

a는 {}로 선언되어 set이므로 중복을 허용하지 않음

즉 kiwi만 추가 banana, apple은 이미 있으므로 변화 없음

</aside>

### 기출 따라잡기 문제 4.py

```java
a = 100
result = 0
for i in range(1,3):
	result = a >> i
	result = result + 1
print(result)
```

```java
# 출력
26
```

<aside>

>>는 비트연산자로 >> 1은 a를 2로 나누는 것과 동일 >> 오른쪽에 있는 숫자가 커질수록 $2^2$, $2^3$, $2^4$,,,로 나눠주면 됨

근데 여기선 result = a >> i 에서 a 때문에 계속 100으로 덮어써지는거 꼭 기억할 것

result = 100 → 50 → 51 → 100 → 25 → 26

</aside>

### 기출 따라잡기 문제 7.py

```python
lst = [1,2,3]
dst = {i : i* 2 for i in lst}
s = set(dst.values( ))
lst[0] = 99
dst[2] = 7
s.add(99)
print(len(s & set(dst.values( ))))
```

```java
# 출력
2
```

<aside>

dst = {i : i* 2 for i in lst} → {{1 : 2},{2 : 4},{3 : 6}}

s = {2, 4, 6}

lst = [1, 2, 3] → [99, 2, 3]

dst = {{1 : 2}, {2 : 4}, {3 : 6}} → {{1 : 2}, {2 : 7}, {3 : 6}}

s = {2, 4, 6} → {2, 4, 6, 99}

s & set(dst.values( ))는 두 변수 간의 교집합을 찾음 즉 {2, 4, 6}과 {2, 7, 6}의 교집합은 {2 ,6}최종적으로 {2, 6}의 길이인 2가 출력
