### [기출 예제] 24년 3회 실기

- **아래 자바 코드에서 출력되는 값을 작성하시오.**

```jsx
public class Main {
		static String[] s = new String[3];
		
		static void func(String[] s, int size) {
				for (int i = 1; i < size; i++) {
						if (s[i - 1].equals(s[i])) {
								System.out.print("O");
						} else {
								System.out.print("N");
						}
				}
				for (String m : s) {
						System.out.print(m);
				}
		}
		
		public static void main(String[] args) {
				s[0] = "A";
				s[1] = "A";
				s[2] = new String("A");
				
				func(s, 3);
		}
}
```

- 정답
    
    OOAAA
    
- 풀이
    - `equals()`  : 문자열 내용 비교
    - `==` : 같은 객체를 가리키는지 비교

### [기출 예제] 21년 1회 실기

- **클래스 내에서 객체 생성 없이 사용할 수 있는 메소드로써 출력 결과를 작성하시오.**

```jsx
public class Main {
		public static void main(String[] args) {
				int i, j;
				for(j = 0, i = 0; i <= 5; i++) {
						j += i;
						System.out.print(i);
						if(i == 5) {
								System.out.print("=");
								System.out.print(j);
						} else {
								System.out.print("+");
						}
				}
		}
}
```

- 정답
    
    0+1+2+3+4+5=15
