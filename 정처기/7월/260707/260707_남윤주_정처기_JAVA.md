# 상속
부모 클래스가 가진 특성(변수)와 행동(메서드)을 자식 클래스가 물려받는 것

```
class Parent {
    void sayHello() {
        System.out.println();
    }
}
class Child extends Parent {

}
```
Child 클래스는 Parent 클래스의 sayHello 메서드를 상속받음.
Child 객체를 만들어 sayHello를 호출하면 메시지가 출력.

## 오버라이딩
상속받은 메서드를 자식 클래스에서 재정의. 
```
class Child extends Parent {
    void sayHello() {

    }
}
```
## 오버로딩
같은 이름의 메서드를 여러 개 가지되, 매개변수의 유형이나 개수를 다르게 하는 것. 
```
class MathOperations {
    int sum(int a, int b) {
        return a + b;
    }
    int sum(int a, int b, int c) {
        return a + b + c;
    }
}
```
MathOperations 클래스에는 sum 메서드가 두 개 있음. 매개 변수가 서로 다르므로 두 메서드는 다른 버전으로 존재 가능.

오버라이딩은 자식 클래스가 부모 클래스의 메서드를 재정의. 오버로딩은 같은 이름 메서드 여러 개.
