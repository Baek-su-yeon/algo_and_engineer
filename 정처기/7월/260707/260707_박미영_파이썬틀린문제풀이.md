### [기출 예제] 21년 2회 실기

- **다음 파이썬 코드에 대한 출력값을 작성하시오.**

```python
a = 100
result = 0
for i in range(1, 3):
		result = a >> i
		result = result + 1
print(result)
```

- 정답
    
    26
    
- 왜 틀렸는가
    
	100을 2진수로 바꾸면 1100100 나오는 것까지는 ok
	
	근데 얘를 오른쪽으로 2번 시프트한다는 걸 그냥 숫자만 옮기면 되는데
	
	8자리를 맞춰야 된다고 생각하고 '0011001' 이 아니라 '00110010' 이라고 생각함.

	8자리를 맞춘다는 생각이 틀린 건 아닌데 그럴려면 왼쪽에 0을 붙였어야 됨.

### [기출 예제] 25년 1회 실기

- **다음은 파이썬에 대한 문제이다. 아래 코드를 확인하여 알맞는 출력값을 작성하시오.**

```python
class Node:
		def __init__(self, value):
				self.value = value
				self.children = []
				
def tree(li):
		nodes = [Node(i) for i in li]
		for i in range(1, len(li)):
				nodes[(i - 1) // 2].children.append(nodes[i])
		return nodes[0]
		
def calc(node, level=0):
		if node is None:
				return 0
		return (node.value if level % 2 == 1 else 0) + sum(calc(n, level + 1) for n in node.children)

li = [3, 5, 8, 12, 15, 18, 21]

root = tree(li)

print(calc(root))
```

- 정답
    
    13
    
- 풀이

	```python
	# 노드를 만들 때 실행되는 초기화 함수
	# value라는 값을 받아서, 노드 안에 저장하고 자식 노드들을 담을 빈 리스트도 함께 가지고 있음.
	class Node:  
			def __init__(self, value):
					self.value = value
					self.children = []
					
	def tree(li):
			nodes = [Node(i) for i in li]  # 리스트 li의 각 값을 이용해 Node 객체들을 순서대로 만듦. ex) [Node(3), Node(5), ..., Node(21)]
			for i in range(1, len(li)):
					nodes[(i - 1) // 2].children.append(nodes[i])  # 인덱스 i의 노드를, 인덱스 (i-1)//2의 노드의 자식으로 연결  => 리스트가 완전 이진 트리로 변경됨.
			return nodes[0]
			
	def calc(node, level=0):
			if node is None:  # 재귀 종료
					return 0
			# 현재 레벨이 홀수면 node.value를, 짝수면 0을 더함.
			# 이 노드 자식들에 대해, (level + 1) 하여 calc를 호출한 결과를 전부 더함.
			return (node.value if level % 2 == 1 else 0) + sum(calc(n, level + 1) for n in node.children)

	li = [3, 5, 8, 12, 15, 18, 21]

	root = tree(li) # li로 트리를 만듦.(아래에 사진 첨부)

	print(calc(root))
	```

dst = {i : i * 2 for i in lst}

print(len(s & set(dst.values())))

### [기출 예제] 26년 1회 실기

- **아래 파이썬 코드가 있다. 입력값으로 HumanDev를 주었을 때 출력되는 결과를 쓰시오.**

```python
i = input()
x = []

for word in i.split():
		x.append(word)
		
y = ''.join(x)
z = ''.join(c for c in y[::-1] if c not in 'ong')

print(z)
```

- 정답
    
    veDamuH
    
- 풀이
    - **`z = ''.join(c for c in y[::-1] if c not in 'ong')`**
        - `if c not in 'ong'` 는 o, n, g에 해당하는 문자는 빼고 골라야 됨.
