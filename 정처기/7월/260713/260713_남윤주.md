## 1.
```
#include <stdio.h>

int main() {
    int a = 6;
    int b = 3;
    int *p = &a;
    int **q = &p;

    *p = (*p ^ b) + (**q & b);

    printf("%d", a);
    return 0;
}
```
출력 결과를 쓰시오.
정답
7
해설
*p와 **q는 모두 a를 의미함.
a ^ b : 2진수로 변환하여 XOR 문 수행
6(110) ^ 3(011) = 5(101)
a & b
6(110) & 3(011) = 2(010)
따라서 5 + 2 = 7

## 2.
문제
```
class Parent {
    static int count = 0;

    Parent() {
        count++;
    }

    void print() {
        System.out.print("P");
    }
}

class Child extends Parent {
    Child() {
        count += 2;
    }

    @Override
    void print() {
        System.out.print("C");
    }
}

public class Main {
    public static void main(String[] args) {
        Parent obj = new Child();
        obj.print();
        System.out.print(Parent.count);
    }
}
```
출력 결과를 쓰시오.
정답
C3
해설
new Child() 실행
자식 생성자보다 부모 생성자가 먼저 실행됨
Parent()에서 count = 1
Child()에서 count = 3
obj의 선언 타입은 Parent지만 실제 객체는 Child
오버라이딩된 Child.print()가 실행되므로 C 출력

문제

정답
SELECT E.name
FROM EMP E
JOIN INCENTIVE I
  ON E.emp_id = I.emp_id
WHERE I.amount >= 500;

## 4.
문제
IP 주소가 다음과 같을 때 네트워크 주소와 브로드캐스트 주소를 쓰시오.
IP 주소: 192.168.35.10
서브넷 마스크: 255.255.252.0

정답
네트워크 주소: 192.168.32.0
브로드캐스트 주소: 192.168.35.255

해설
255.255.252.0은 세 번째 옥텟 기준 블록 크기가 256 - 252 = 4
네트워크 범위는 다음과 같음.
192.168.32.0 ~ 192.168.35.255
따라서 네트워크 주소는 192.168.32.0
마지막 주소인 브로드캐스트는 192.168.35.255
