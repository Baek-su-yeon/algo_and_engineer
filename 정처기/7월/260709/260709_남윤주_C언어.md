# 포인터
메모리 주소를 저장하는 변수. 특정 데이터의 주소를 가리킴. 직접적으로 메모리에 접근하고 데이터를 효율적으로 관리할 수 있음. p = &a; 는 p가 변수 a의 주소를 가리키며 *p로 a의 값에 접근.

# 포인터 선언
어떤 타입의 주소를 저장할지 명시하고 변수명 앞에 애스터리스크*를 붙여줌
char *p; 문자열의 주소를 저장할 포인터 p

# 개념
기본적으로 문자열 "Art"가 있다면 이 문자열에는 \n까지 총 4개의 문자가 들어있음. %s는 문자열 출력 형식 지정자, %c는 단일 문자 출력 형식 지정자.

# 예제
```c
char a[] = "Art";
char* b = "Good";

printf("%s", a) // Art
printf("%c", a[0]) // A
printf("%c", *a) // A
printf("%c", *(a+2)) // t
printf("%s", a+1) //rt

printf("%s", b) //Good
printf("%s", b+1) // ood
printf("%c", *b) //G
printf("%c", *(b+3)) //d

printf("%c", *b+3) //J
```
b라는 포인터 변수는 "Good" 문자열 전체가 아닌 첫 글자 "G"의 메모리 주소를 가리킴.
