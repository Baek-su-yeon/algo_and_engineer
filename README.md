## 개발 취준 루틴 스터디

### 참여자
> 남윤주, 박미영, 백수연

### 목표
> 개발 직군 취업 및 꾸준한 학습 루틴 형성
---
### 활동
|요일|학습 내용|
|:---|:---|
|**월·수·금**|알고리즘 문제 풀이|
|**화·목**|정보처리기사 또는 CS 내용 정리|

### 활동 방법
1. 각자 공부 브랜치 생성
    > 브랜치명은 알아서 편한대로 생성
2. 요일별 학습 진행
3. 공부일자에 맞는 폴더에 오늘 학습 내용 정리 파일 업로드
    > 파일명은 "날짜_본인이름_알고문제이름", "날짜_본인이름_학습내용"으로 고정
4. PR 생성 후 팀원들의 리뷰를 받고 Merge
5. **🎊 오늘 학습 완료 🎉**
---
### 폴더 구성
```
archive // 이전에 풀었던 알고리즘 문제 모음, 아무나 확인 가능
 ㄴ📂문제번호_문제이름 (예:1949_등산로조성)
   ㄴ📄문제_풀이_코드
   ⁝

알고리즘
 ㄴ📂해당일자 (예:260615)
   ㄴ📄각자_문제_풀이_코드 (예:260615_백수연_등산로조성)
   ⁝

정처기
 ㄴ📂해당일자 (예:260615)
   ㄴ📄각자_정리_내용 (예:260615_백수연_시나공코드기출풀이)
   ⁝
```

- 학습 내용 업로드 전 **git pull** 꼭 받기
- 다른 일정으로 인해 그날 학습을 진행 못할 것 같을 때는 **미리 진도 빼기**
- 아예 못하는 경우 **디코에서 하루동안 얼굴 캠키고 공부 진행📺**
